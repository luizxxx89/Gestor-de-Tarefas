# Requisitos Não Funcionais

## Projeto

Sistema de Gerenciamento de Tarefas

---

# RNF01 - Usabilidade

A interface deverá ser simples e intuitiva, permitindo que usuários utilizem o sistema sem dificuldades.

---

# RNF02 - Responsividade

O sistema deverá funcionar corretamente em computadores, tablets e smartphones.

---

# RNF03 - Desempenho

As operações principais do sistema devem apresentar resposta rápida ao usuário.

---

# RNF04 - Segurança

O sistema deverá proteger os dados dos usuários e armazenar senhas de forma segura.

---

# RNF05 - Disponibilidade

O sistema deverá estar disponível sempre que o usuário precisar acessar suas tarefas.

---

# RNF06 - Manutenibilidade

O código deverá ser organizado seguindo boas práticas de desenvolvimento, facilitando futuras alterações.

---

# RNF07 - Tecnologias

O projeto deverá utilizar:

## Front-end

- React
- JavaScript
- HTML
- CSS

## Back-end

- Node.js
- Express

## Banco de Dados

- SQLite (versão inicial)

## Versionamento

- Git
- GitHub
