# Requisitos Funcionais

## Projeto

Sistema de Gerenciamento de Tarefas

---

# RF01 - Cadastro de Usuário

O sistema deverá permitir que o usuário realize cadastro informando:

- Nome.
- E-mail.
- Senha.

---

# RF02 - Login

O sistema deverá permitir autenticação do usuário através de e-mail e senha.

---

# RF03 - Criar Tarefa

O usuário deverá conseguir cadastrar uma nova tarefa contendo:

- Título.
- Descrição.
- Data de criação.
- Status.

---

# RF04 - Listar Tarefas

O sistema deverá apresentar todas as tarefas cadastradas pelo usuário.

---

# RF05 - Editar Tarefa

O usuário deverá conseguir alterar informações de uma tarefa existente.

---

# RF06 - Excluir Tarefa

O usuário deverá conseguir remover uma tarefa cadastrada.

---

# RF07 - Concluir Tarefa

O usuário deverá conseguir alterar o status de uma tarefa para concluída.

---

# RF08 - Filtrar Tarefas

O sistema deverá permitir visualizar tarefas por status:

- Pendentes.
- Concluídas.
